A · 左边栏 + 实底色★推荐：竖条边界 + 明显底色 + 语言标签，和 blockquote 风格统一，窄屏稳

下面是示例：

js

▏ export function add(a, b) {
▏   return a + b;
▏ }

就这样。

B · 圆角边框 + 标题最精致：语言名嵌在顶边。占 2 列，窄屏/长行换行时易错位

下面是示例：

╭─ js ──────────────────────────╮
│ export function add(a, b) {       │
│   return a + b;                   │
│ }                                 │
╰───────────────────────────────────╯

就这样。

C · 行号栏 + 底色便于指代“第几行”，适合长代码、教学；行号占宽

下面是示例：

js

  1 │ export function add(a, b) {
  2 │   return a + b;
  3 │ }

就这样。

js
